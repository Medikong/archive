# 플랫폼 운영 사용자 목표

## 기본 정보

- UC ID: `UC.A.03`
- 사용자: 운영 관리자, 상품/콘텐츠 검수 담당자, 제휴/이벤트 담당자, 쿠폰 운영 담당자, 정산 담당자, 인시던트 담당자
