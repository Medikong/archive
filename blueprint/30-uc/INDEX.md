# 유스케이스 인덱스

## 액터 정의

| 액터 | 정의 |
| --- | --- |
| 비회원 | 인증되지 않은 사용자. 공개 탐색, 로그인, 회원가입, 비밀번호 재설정을 할 수 있다. |
| 구매자 | 인증이 완료되어 구매자 권한을 가진 사용자. 장바구니, 주문/결제, 마이, 쿠폰함, 주문 내역, 배송 조회를 사용할 수 있다. |
| 판매자 | 판매자 포털에서 상품, 드롭, 쿠폰, 주문/출고 자료, 성과를 관리하는 사용자. |
| CS 담당자 | 문의 처리를 위해 사용자, 주문, 결제, 쿠폰, 배송 상태를 조회하고 처리 이력을 남기는 사용자. |
| 플랫폼 운영자 | 플랫폼 정책, 검수, 쿠폰, 인시던트, 위험 작업을 관리하는 사용자. |
| 외부 시스템 | DropMong 밖에서 결제, 배송, 알림, 인증 같은 기능을 제공하거나 연동되는 시스템. |
