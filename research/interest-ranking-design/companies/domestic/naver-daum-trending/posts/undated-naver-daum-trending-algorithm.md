# 네이버 실시간 급상승 검색어 알고리즘 / KISO 검증보고서

- 회사: 네이버, 다음
- 원문: https://www.clien.net/service/board/park/5599879 , https://www.kiso.or.kr/wp-content/uploads/2014/10/3%EC%B0%A8-%EA%B2%80%EC%A6%9D%EB%B3%B4%EA%B3%A0%EC%84%9C.pdf , https://journal.kiso.or.kr/?p=9967
- 접근일: 2026-07-09
- 주제: 급상승 판정 통계 방식, 알고리즘 신뢰성 검증
- 확인 방식: WebSearch 결과 스니펫 기준 (커뮤니티 설명 글 + KISO 공식 검증보고서 존재 확인, 보고서 원문 전체는 미열람)

## 원문 확인 내용

- 절대적인 검색량이 아니라 검색 빈도 비율이 얼마나 상승했는지를 기준으로 급상승 검색어를 선정한다.
- 짧은 시간(약 15초) 동안 입력되는 질의 수를 최근 약 10분간의 평균값·표준편차와 비교해, 차이가 큰 키워드를 추출한다.
- 실시간 급상승 키워드 추출 시스템은 일정 간격마다 검색 서버 로그를 수집해 실시간으로 분석하고, 현재 데이터와 기준 데이터를 비교해 급격히 증가한 키워드를 뽑아낸다.
- 다음 소셜픽은 검색어 쿼리 검색량/주기성뿐 아니라 뉴스·트위터의 댓글·리트윗 논의량까지 함께 고려한다는 점에서 네이버와 차이가 있다.
- 네이버는 2021-02-25부터, 다음은 2020-02-20부터 서비스를 중단했다가(다음은 2026-03-03 재개) — 조작 논란과 사회적 신뢰성 문제가 배경으로 언급된다.

## 적용 아이디어

- "최근 평균 대비 증가율" 방식은 DropMong이 향후 "지금 급상승 중인 드롭" 같은 별도 신호를 만들 때 참고할 통계적 틀이 된다.
- 알고리즘 신뢰성이 사회적 검증 대상이 될 정도로 예민하다는 사실은, `REQ.A.07.NFR-003`(조회수 중복 방지는 완전한 봇 탐지가 아님을 명시)처럼 DropMong도 랭킹 로직의 한계를 문서에 명확히 남겨야 한다는 근거로 쓴다.

## 확인 필요

- KISO 검증보고서 원문(PDF) 전체는 열람하지 않았다. 구체적 통계 임계치나 검증 방법론은 원문을 직접 읽어야 확인 가능하다.
