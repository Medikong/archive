# Research

외부 기업 사례, 공개 기술 문서, 제품 설계 참고 자료를 장기 보관하는 조사 영역이다.

이 영역은 Medikong/DropMong 내부 설계 문서와 분리해서 관리한다. 원문 스크랩, 출처 목록, 회사별 정리, 2차 분석을 함께 두되, 실제 제품 설계 결정은 각 설계 문서나 ADR에서 별도로 기록한다.

## 조사 묶음

| 주제 | 용도 | 시작 문서 |
| --- | --- | --- |
| `auth-service-design/` | 국내/해외 기업의 인증 서비스 설계 사례 조사 | [auth-service-design/README.md](auth-service-design/README.md) |

## 작성 기준

- 출처 URL, 접근일, 저장 위치를 남긴다.
- 원문 스크랩과 해석/분석 문서를 분리한다.
- 공개 자료에서 확인한 내용과 추정한 내용을 구분한다.
- 회사별 자료는 `companies/`, 주제별 재정리는 `analysis/`에 둔다.
